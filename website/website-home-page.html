CREATING A WEBSITE - BEGINNERS GUIDE

Welcome

Want to create a website? dont know where to start? This webiste will help you understand the fundamentals of creating a website.

The intial understanding of where a website begins with knowing how to write,understand and parse through laguages such as HTML, CSS, Javascript. Effectively being confident in all three languages is most important as anyone can copy and paste code, however, this will eventually catch up with them as they wont be able to interpret their own code, and will get caught going round in circles within the three languages.

Work your way through this webstie, beginning with HTML, which is used for the structuring of the web and contents within a web page.

Then move on to CSS, which is a stylesheet language used for the desgin and presentation of a document in HTML or XML

After CSS, move onto Javascript, which is a the scripting language for web pages, and is used to design the behaviour of web pages when an event takes place.
